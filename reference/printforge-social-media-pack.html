How to Use in Canva

  1. Open this HTML file in Chrome at 100% zoom (or 200% for retina-quality).
  2. Use a screenshot tool (Cmd+Shift+4 on Mac, Snipping Tool on Windows) to capture each asset individually.
  3. In Canva, create a new design at the correct dimensions (listed above each asset).
  4. Upload your screenshot and place it as the background.
  5. Optionally overlay editable Canva text if you want to tweak the copy.
  6. For the Facebook Cover, replace the "QR Code Goes Here" box with a real QR code pointing to crm.printforge.com.au.
